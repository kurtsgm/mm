## 0. 已拍板的三個框架岔路（設計前提）

使用者已就三個載重決策定調：

1. **世界異變成因＝核心「真的遺失在民間」**——不是「一直在眠龕、只是調節退化」。詳見 §1 的自圓機制。
2. **前期定位＝在地英雄漸親帝國**——保留現有橡鎮／四野開場大架構，把反抗（燼火／多恩）入口往後撤或改中性；玩家因平亂護民被帝國／教會賞識、入體制。
3. **反抗線時點＝抉擇後才展開**——前中期純帝國視角（反抗軍是你在剿的對象、看似暴民）；只有「打開她＋不帶回」後，才由領航者帶你真正認識反抗軍、翻轉「誰才是壓迫者」。抉擇的重量來自「未知的好奇 vs 已知的安穩」，不是「兩邊都玩過再選」。

**核心主題不變**：`求真招險換自由（真相線）` vs `埋真苟活保安穩（帝國線）`。本重構只是把這對主題**凝縮成一個極具象的動作——你要不要打開那個盒子看一眼**。不看的人得到舒適的謊言與牢籠；看的人踏上危險的真相。

## 1. 世界觀微調：異變成因（解「遺失在民間」的自圓）

### 1.1 命脈＝搖籃機制（新增設定，補進 bible §10.1／reference G）

- 領航者（核心）以**場能**供養全船（維生·生態·氣候·頭頂那顆「太陽」）。但她那口壓縮恆星的**封印場必須坐在「至聖搖籃／插座」裡才能穩定調節**——這口搖籃＝canon 的**眠龕（Reliquary of Slumber）**，方舟深處的核心插座。
- 四萬年來她沉睡在搖籃裡穩定供電（呼應 canon「沉睡時仍被動供電、穩定維繫全船」），只有極緩慢的老化退化。
- **離座＝失穩**：她一旦離開搖籃，封印場的即時調節失去她這具活體守護者坐鎮，急遽惡化。（接上威脅模型 spec「會退化的是『調節』」——調節者就是她，離座則調節崩壞加速。）

### 1.2 近期竊案（開場前不久／約一代人前）＝把 canon「聖髑突襲」挪來當背景

- **燼火**受**啟明會**點撥，以為帝國深藏一件「權力之源／神祕武器」，突襲**翠淵眠龕**、奪走封印聖髑。（沿用 canon §I.1「燼火欲毀之」動機，只是從中期事件改成**開場前的背景事件**。）
- 奪走後在混亂中**失落**——信使被劫／細胞被剿／落入盜遺者與碎島海黑市。**教會與反抗軍現在都沒有她。她＝真的「流落在外」。**
- **後果**：核心離搖籃 → 封印場失穩、供電調節急遽惡化 → **天燈將熄**（開場正在惡化的世界危機：日照變短、外圈失溫、饑荒、末日教派——直接吃 draft §A 走樣神話「天燈將熄＝末日凶兆」）。

### 1.3 教會的解讀是「預言」不是「知識」（保住 canon 鐵則）

- **canon 鐵則不破**：沒有人知道她是命脈核心——連教會也不知道（bible §0／§10.1）。教會把她當「一件受保護的聖遺物」。
- 教會透過 draft §A 走樣神話「**持光聖女沉睡預言**」（「當天燈將熄，長眠的持光聖女將甦醒，指引子民」）來讀天燈將熄：**聖物被竊、聖光將滅，必須尋回持光聖女、送回至聖所，聖光乃復**。他們是對的（送回搖籃＝復電），卻出於宗教理由、完全不懂機制。**這條走樣預言成為主線扳機。**

### 1.4 王道紅利（why 前期正義看似在帝國這邊）

- 從帝國／百姓視角，這是「**異端盜賊（燼火）偷走了聖光、害世界快死**」。所以前中期玩家剿叛軍＝師出有名、正義看似在帝國這邊——完美支撐「前期王道、不早接反抗」（§0 之 2、3）。
- **叛逃後的翻轉料**（§4a 用）：燼火當初之所以去偷，是因為他們（經啟明會）**懷疑那是帝國暗藏的權力祕密／假神之證**——他們比誰都接近真相。這層在玩家叛逃、知道真相後才回頭補上，把「害世界的罪犯」翻轉成「唯一沒被謊言騙倒的人」。

### 2.1 待拍板的選角／細節（本 spec 的預設，可於審核時調整）

## 4. 兩塊發想（使用者指定）

### 4a. (B2) 反抗軍線的合理串接（連接組織核心）

1. **一叛成通緝**：你沒帶她回去（或被發現打開／帶走她），瞬間成帝國頭號要犯——「**偷走了能救世界的聖光**」。百姓恨你（他們以為你扣住救贖，而天燈仍在熄）＝殘酷反諷＋巨大壓力。
2. **領航者翻轉認知**：她誠實但打啞謎（記憶錯亂）地逐步揭示——帝國要的「救贖」＝把她永遠封回去當電池、保一座無知牢籠；真相在外（遺跡＝飛船、皓月＝故鄉墳、怪＝血親）；真有活路（回泰拉、逃離死星系）。她誠實藏著的獻祭代價（點火燒盡自己）**暫不說**（canon §H 鐵則：終章點火前只給伏筆、不點破）。
3. **天然盟友＝當初的「盜賊」**：你回不了頭，唯一會接納你的，正是帝國口中的異端罪犯——**燼火與啟明會，也就是當初偷走她的人**。串接的漂亮處：反抗軍不是天上掉下來的盟友，而是「早就看穿帝國祕密、試圖奪取／揭露、被打成恐怖分子」的人。**你打開盒子、知道真相後，才懂他們當初為何那麼做。** 領航者＋真相＝把「害世界的罪犯」翻轉成「唯一沒被謊言騙倒的人」的橋。
4. **王道全被重新定義**（呼應 §0-3）：你前中期剿的「叛軍」、掙的帝國信任，回頭看全是在維護牢籠。反抗軍線＝護送領航者、與燼火／啟明會合流、拼出全部真相（聖城半揭 → 泰拉全揭）、走向逃離。**你走上反抗之路不是被早早招募，而是真相讓你無從選邊。**

> **與 canon §E 真相里程碑的對接**：前三塊伏筆（淺層船艙「這石頭不像石頭」／沉艙「這是機器不是墓」／真實歷史殘檔）在幕一～二**零星浮現**（玩家甚至不以為意，或以帝國視角誤讀）；喚醒（開盒）後由領航者把散落線索**一次收束**，接聖城半揭 → 泰拉全揭。順序不變、只重掛在五幕上。

### 4b. (A / B1) 假 happy ending 後續（帝國線·牢籠長存）

- **天燈復明**：送回聖物、聖光重穩、饑荒止、異變逆轉。你是英雄「尋回持光聖女、拯救世界的人」。凱旋、受封、晉身帝心（canon §I.4 晉身）。
- **平叛累積聲望**：危機解除後帝國清算。剩餘工作＝一個個掃——燼火（被歸咎盜聖光）逐細胞獵殺、啟明會肅清、庇護反抗的離心領主（哈洛／瑞德蒙）整肅或清洗（canon §I.6 帝國線關係演變：玩家 × 燼火／啟明會 ○→⊗ 獵殺、玩家 × 哈洛 △→✕ 清洗）。每役抬升地位 → 新晉審判長／皓月之子近臣。**玩法＝一串「拔據點／獵首領」聲望任務。**
- **「假」的那一刀（戲劇反諷，只給玩家、不給角色）**：聖物被重新封存、無人再問；走樣神話更硬化；天燈「修好」了，但**埋伏筆——它偶爾又微微閃爍**（調節只是被打補丁、退化仍在；牢籠買的是時間不是未來）。收尾一擊（canon §I.4 最後一刀）＝片尾對玩家揭露角色永遠不會知道的事：你拚死守護、好讓大家繼續膜拜的皓月，正是自己凍死故鄉的墳；你的「救贖」是把一個人送回去永遠燒著；你蓋了一座餓不死、也活不起來的牢籠。**安穩卻沉淪。**
- **可選 beat 讓 happy ending 發酸**：加冕慶典萬眾歡呼你為救主，對玩家穿插真相一瞥（守燈人被拖走前低語一句／聖髑聖龕重新祝聖那一格畫面看起來像棺材）。角色在笑，玩家發寒。
- **B1（開了、明知、仍帶回）差別**：機制同 A，但你「知情」。反諷更利、更私人——你看著她的眼睛、她告訴你真相，你仍選了牢籠。她不怨（canon 無背叛），留一句誠實而寬恕的話再度長眠。獨立成結局槽「知情的牢籠／共犯的安穩」。
