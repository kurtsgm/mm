**Goal:** 把 mm 主線 canon 從「三段式（中立漫遊 → 反抗側突襲喚醒 → 雙面人雙線 → 後期分歧）」重構成「五幕王道收束 → 抉擇＝要不要打開封印聖遺物」，並讓兩本 canon 檔（`world-bible.md`／`world-reference.md`）與既有遊戲 content 全面一致。

**Architecture:** 這是**canon 敘事文件的結構性重寫**，不是程式碼。權威來源＝已提交的 spec `docs/superpowers/specs/2026-07-02-王道帝國線與開盒抉擇重構-design.md`。工作＝依 spec 逐「敘事關切」改兩本 canon＋既有 content JSON；每個 task 的「測試」＝**一致性 grep 掃描（舊結構詞已重構乾淨）＋讀回校對（無互相矛盾）**，content task 另跑 quest 靜態驗證器與 boot smoke。

**Tech Stack:** Markdown（canon 檔）、JSON（`content/`）、Godot 4.7 headless（`tools/quest_lint_cli.gd` 驗證器、`./run.sh --headless` boot smoke）。

## Global Constraints

- **權威來源＝committed spec**：所有新 canon 內容的細節以 `docs/superpowers/specs/2026-07-02-王道帝國線與開盒抉擇重構-design.md` 為準；本 plan 指到 spec 某節時，該節內容即該步驟要落的料。
