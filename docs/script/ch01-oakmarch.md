## 0. 章節設計原則（先讀）

1. **在地小事為主軸。** 主線是橡境的在地麻煩（哥布林、領主苛政、鄉里民生），**與真相無關**（守 bible §設計原則 2）。
2. **帝國／反抗只是「初遇」。** 依 §9.1 Phase 1「中立漫遊、帝國與反抗背景偶現」：卡西安過境＝玩家對審判庭的**第一次照面（寒意種子）**；多恩＝對燼火的**第一次接觸（開一道門）**——都不是全面反抗戰役（那是 Phase 2）。
3. **讓玩家先恨上壓迫秩序。** 透過苛政與審判庭的冷酷，建立情感動機；但**領主也是夾心**，把矛頭指向更上面。
4. **暗線只給伏筆、不點破。** 領航者殘響、枯石谷遺跡「這石頭不像石頭」——說不上哪裡不對，純鋪陳（守 §設計原則 3）。
5. **人物行動一律走 A.9 決策模型**，確保反應合理、可預測。
6. **無硬分支。** 大抉擇在 Phase 3；本章玩家的選擇只影響**風味與聲望**（如何應付卡西安、幫不幫鄉里），不分岔主線。

## 1. 章節目標（本章結束時要達成）

- 玩家熟悉 hub（橡鎮：招募／療癒／補給／法術）與冒險循環（接懸賞→出野外→回報）。
- 建立五張人臉：哈爾（守衛隊長）、瑪歌（療者盟友）、哈洛（離心領主樞紐）、卡西安（審判庭寒意）、多恩（燼火入口）。
- 種下兩道暗線伏筆：**領航者殘響**（B2、B7）、**枯石谷淺遺跡**（B7）。
- 讓玩家對帝國—神教壓迫秩序產生情感反感。
- 章末給出「**深入內地（鐵橋城）**」的理由與鉤子，銜接第二章。

## 4. 任務鏈 beat 大綱

> 每個 beat：**前置 → 內容 → 人物動作（決策模型）→ 綠定 → 伏筆 → 過關條件**。

### 幕一　落腳與在地小事

#### B1　抵達橡鎮（hub 建立）
- **前置**：新隊伍（旅店/公會招募的冒險者）循懸賞抵達橡境。
- **內容**：入 `town_oak`。認識 hub——旅店（招募/休憩）、神殿（瑪歌療癒）、商店、法師塔。城門/告示板有哥布林懸賞。
- **人物動作**：守衛隊長**哈爾**在城門打量來客、引導去看告示板（直爽老兵口吻）。
- **綠定**：`qg_oak_guard` 賦名哈爾；vendor 即現有商店/神殿/法師塔。
- **過關**：玩家接下哥布林懸賞（進 B2）。

#### B2　哥布林的威脅（主線開場＋伏筆 A）
- **前置**：接下懸賞。
- **內容**：東北野巢穴清哥布林 → 取信物 → 確認最深處 → 回橡鎮向哈爾回報（即現有 `goblin_menace` 四階段）。
- **人物動作**：哈爾各階段催進度/道謝（現有對白即可）。
- **綠定**：`goblin_menace.json`、`qg_oak_guard.json` 原樣沿用。
- **伏筆 A（領航者第一道殘響）**：在「確認巢穴最深處」（`wild_ne` 深處格）安排一截**半埋的金屬物**（淺遺跡碎片）；玩家靠近時觸發**領航者早期殘響**（遺物殘響載體，H.5「封印殘響」口吻，如「……你聞到鐵鏽了嗎——那不是血。是門。」）。**極輕**點一句哥布林「那張臉，怎麼有點像人」的不安（§4 第3類，不解釋）。
- **過關**：回報完成 → 解鎖哈洛接見（進 B4）；瑪歌可在回程觸發（B3）。

#### B3　瑪歌登場（療者盟友）
- **前置**：B2 進行中或回程（可與主線並行）。
- **內容**：神殿/鎮上，瑪歌替傷者或玩家療傷，露出邊陲民生的苦難；給一個**可選小支線**（去東南野毒澤採藥／救治一名被苛稅逼到流亡的病患）。
- **人物動作**：瑪歌**無條件先治、帳後算**（A.9.5），對苦難流露同情、但不選邊。
- **綠定**：神殿 vendor 的具名療者人臉＝瑪歌；採藥支線可用 `wild_se`。
- **過關**：認識瑪歌（盟友/療癒據點建立）；支線可選完成。

### 幕二　苛政與審判庭的陰影

#### B4　領主苛政 vs 鄉里
- **前置**：B2 回報後解鎖。
- **內容**：哈洛（經哈爾引見或接見）拋出為難事——帝國加派的徵糧/苛稅壓下來，鄉里快撐不住。玩家替他緩解：智取/勸退一個帝國稅吏，或平息因苛稅而起的鄉里糾紛。
- **人物動作**：哈洛**生存優先、陽奉陰違**（A.9.2）——表面配合帝國、暗中要玩家替鄉里解圍；點明「領主也是夾心，真正壓在頭上的在更上面」。
- **綠定**：新增哈洛對話（暫 `qg_oak_lord`）＋苛政小任務（暫 `oak_levy`）。
- **過關**：緩解苛政 → 掙得哈洛初步信任。

#### B5　卡西安「灰燭」過境（審判庭寒意）
- **前置**：B4 後，鎮上風聲鶴唳。
- **內容**：審判官卡西安帶沉默武僧過境橡鎮緝異端。盤查鎮民、點名一個「可疑者」（建議＝瑪歌救治的某流民，或被疑私藏「遺物」的鎮民）。玩家目睹審判庭的冷酷無情。
- **人物動作**：卡西安**禮貌、輕聲、漸進佈線、不打草驚蛇**（A.9.1）；對哈洛＝表面客氣、暗探把柄（卡西安 ✕ 哈洛）。哈洛**暗中拜託玩家拖住/誤導審判官**（A.9.2 暗裡阻撓）。
- **小選擇（風味，不分岔）**：玩家可選**配合哈洛拖延** / **明哲保身旁觀**——影響聲望與卡西安對你的標記，不改主線。
- **綠定**：新增卡西安過境事件/對話（暫 `oak_inquisitor`）。
- **伏筆**：卡西安若察覺玩家碰過「遺物」（B2 殘響物）→ 不發作，**記下你**（A.9.1 ①），為後續埋線。
- **過關**：過境事件落幕（可疑者被帶走/被救/逃脫），玩家對壓迫秩序生厭 → 被引向多恩（B6）。

#### B6　初遇燼火（多恩，反體制初接觸）
- **前置**：B5 的清查餘波。
- **內容**：因審判庭清查，玩家被引向鐵匠多恩。多恩先**試探玩家人品**；玩家通過（藏匿一名被審判庭盯上的人／替他跑一趟祕密口信）→ 正式接觸燼火，第一次站到體制對立面。
- **人物動作**：多恩**裝傻、先試探立場與可靠度**（A.9.3 ①），通過後**逐步給敏感任務**（A.9.3 ②）。
- **綠定**：**「東北野的口信」重定位為多恩的燼火跑線**——信使（`qg_nw_messenger`@`wild_nw`）→斥候（`qg_ne_scout`@`wild_ne`）＝多恩下線；玩家替他跑＝不知情地替反抗組織傳情報。可在對白埋一句「這口信……你最好別拆開看」。新增多恩對話（暫 `qg_dorn`）。
- **過關**：通過多恩試探 → 燼火這道門打開（種子已埋，深入留待第二章）。

### 幕三　伏筆與離境

#### B7　枯石谷的異樣（伏筆 B）
- **前置**：可掛在某任務（多恩/哈洛差事）或自由探索西南野。
- **內容**：西南野枯石谷的**淺層遺跡**——半埋艙壁、非自然的直角結構、「這石頭不像石頭」。靠近時**領航者殘響再起、比 B2 更清晰一點**（設備雜訊/遺物殘響載體）。純伏筆、說不上哪裡不對。
- **人物動作**：（領航者無實體，僅殘響）。
- **綠定**：`wild_sw` 現有**夢魘妖遭遇**結合進來（夢魘妖＝純本地靈能精怪，靠近遺跡靈能外滲強，reference F）；新增枯石谷遺跡事件（暫 `sw_ruin_echo`）。
- **伏筆**：遺跡＝飛船的**最早邊緣暗示**（reference E 漫遊·起點列）；不解釋、不揭露。
- **過關**：撞見遺跡＋殘響 → 暗線深植玩家心裡（無顯性任務獎勵亦可）。

#### B8　清洗逼近、離境鉤子（章末過渡）
- **前置**：B6（接觸燼火）＋ B5（被卡西安記下）後。
- **內容**：卡西安的清查升級（或上報引來更大審判庭壓力），橡境變得太燙。多恩/哈洛的線同時指向**往內地鐵橋城**——燼火需要有人把某物/某口信帶往內陸的更大據點；或玩家因協助反抗已被審判庭標記、**必須離開橡境**。
- **人物動作**：哈洛**權衡後送玩家走、留條後路**（A.9.2）；多恩**引介上層、給跨地差事**（A.9.3 ②）；卡西安**升級為纏上來的獵手**（A.9.1 ③ 的前奏）。
- **綠定**：新增離境事件/任務（暫 `oak_departure` → 指向 `鐵橋城`，reference B.4）。
- **過關**：隊伍**掙得且被迫深入內地** → **第一章結束**，銜接第二章（離心邊境領主領地·鐵橋城）。

- **接點**：第二章開場可由多恩的上線/哈洛的引介，把隊伍送進鐵橋城的政教角力。
